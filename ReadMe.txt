파이썬이 아예 설치되어 있지 않은 새로운 PC에서 업무 자동화 환경을 세팅하기 위해 필요한 전체 과정을 순서대로 깔끔하게 정리해 드립니다.

1단계: 파이썬(Python) 설치하기 (가장 중요)
아무것도 없는 PC라면 가장 먼저 파이썬을 설치해야 합니다.

파이썬 공식 홈페이지(python.org)에 접속하여 최신 버전의 설치 파일을 다운로드합니다.

설치 파일을 실행한 후, 화면 하단에 있는 Add python.exe to PATH (또는 Add Python to PATH) 체크박스를 반드시 클릭한 후 Install Now를 눌러 설치합니다. (이 체크를 누락하면 2단계의 설치 명령어가 작동하지 않습니다.)

2단계: 필수 라이브러리 설치하기
파이썬 설치가 완료되면, 화면 하단의 윈도우 검색창(윈도우키 + R)에 **cmd**를 검색하여 **명령 프롬프트(터미널)**를 실행합니다. 나타난 검은 창에 아래의 명령어를 그대로 복사하여 붙여넣고 엔터를 치면 필요한 모든 라이브러리가 한 번에 설치됩니다.

python -m pip install streamlit pandas python-pptx pillow moviepy pywin32 openpyxl numpy

혹은 requirements.txt 파일이 있다면

python -m pip install -r requirements.txt


위 설치가 모두 끝나면, 
작업하고자 하는 폴더에
"app.py", "프로그램실행.bat" 2개 파일 복사, 붙여넣기 후 
프로그램실행.bat을 더블클릭하여 정상적으로 프로그램을 사용하실 수 있습니다!


현재 작성된 코드의 Standard (16:9 가로) 영상 제작 로직은 가로 사진과 세로 사진이 섞여 있을 때 아주 스마트하게 작동하도록 설계되어 있습니다!
핵심 원리는 "세로 사진 대기열(Queue)" 시스템입니다. 코드는 파일 이름 순서대로 사진을 하나씩 확인하며 다음과 같이 처리합니다.

⚙️ 가로/세로 혼합 시 작동 원리
가로 사진을 만났을 때:
대기하지 않고 즉시 16:9 화면에 꽉 차게 1장으로 영상을 만듭니다.

세로 사진을 만났을 때:
화면의 절반만 차지하므로 바로 영상을 만들지 않고 '대기열'에 잠시 보관합니다.

가로와 세로가 섞여서 들어오는 경우 (예시):
1번 사진 (가로): 즉시 1장으로 화면에 띄움.
2번 사진 (세로): 혼자 화면을 채울 수 없으니 대기열에 들어감.
3번 사진 (가로): 대기열에 있는 2번 세로 사진을 무시하고, 즉시 1장으로 화면에 띄움. (2번 세로 사진은 여전히 대기 중)
4번 사진 (세로): 세로 사진이 들어왔으므로, 아까부터 대기 중이던 2번 세로 사진과 짝을 지어 2장을 나란히 화면에 띄움.

마지막에 세로 사진이 1장 남는 경우 (홀수 개일 때):
모든 사진을 다 처리했는데 세로 사진이 1장 남아있다면 빈칸으로 둘 수 없으므로, 영상에 쓰였던 "첫 번째 세로 사진"을 다시 불러와서 마지막 세로 사진과 짝을 지어 화면을 꽉 채우고 마무리합니다. (세로 사진이 총 1장뿐이라면, 똑같은 사진 2장을 좌우로 나란히 쌍둥이처럼 배치합니다.)

💡 요약
파일이 어떻게 섞여 있든 상관없이 가로 사진은 나오는 즉시 1장씩 단독으로 보여주고, 세로 사진은 나오는 순서대로 2장씩 차곡차곡 모아서 짝을 지어 보여주도록 깔끔하게 자동 정렬됩니다. 파일 이름을 억지로 맞추거나 가로/세로를 따로 분류하실 필요가 전혀 없습니다!

📦 설치되는 라이브러리별 역할 (참고사항)
명령어에 포함된 각 패키지들은 제작하신 프로그램에서 다음과 같은 역할을 수행합니다.
 - streamlit: 프로그램을 실행할 때 인터넷 브라우저에 예쁜 조작 화면(UI)을 띄워줍니다.
 - pandas & openpyxl: 엑셀 파일(.xlsx)에 적힌 강아지들의 정보를 읽고 가져옵니다.
 - python-pptx: 엑셀에서 가져온 텍스트를 파워포인트(PPT) 템플릿의 알맞은 위치에 채워 넣습니다.
 - pywin32: 윈도우에 설치된 파워포인트 프로그램을 직접 조종하여 사진을 알맞은 크기와 위치에 삽입하고, 빈 슬라이드를 삭제합니다.
 - pillow (PIL): 사진의 크기를 조절하고, 세로형 사진 2장을 이어 붙이거나 사진 위에 텍스트 자막을 그립니다.
 - moviepy & numpy: 사진들을 부드러운 전환 효과(크로스페이드)와 함께 이어 붙이고, 배경음악(BGM)을 덧씌워 최종 MP4 동영상 파일로 만들어냅니다.
